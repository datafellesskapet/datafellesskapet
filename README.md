# Datafellesskapet

## Development

Based on [VuePress](vuepress.vuejs.org).

### Prerequisites

- [Node.js v12.x:](https://nodejs.org)
- [Yarn](https://yarnpkg.com)

### How to run VuePress

Install the dependencies of project with:

```bash
yarn install
```

Start a local VuePress development server with:

```bash
yarn dev
```
